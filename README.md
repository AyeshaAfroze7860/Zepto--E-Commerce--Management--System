###Zepto E-Commerce Management System

##Project Overview

The Zepto E-Commerce Management System is a Database Management System (DBMS) project developed to manage the operations of an online grocery shopping platform. The system enables customers to browse products, place orders, make secure payments, and track deliveries. It also provides administrators with tools to manage products, inventory, customers, orders, and reports efficiently.

##Features
  Customer registration and login
  Product browsing and search
  Shopping cart management
  Online order placement
  Secure payment processing
  Real-time order tracking
  Inventory management
  Admin dashboard for managing products and orders
  Sales and inventory report generation
